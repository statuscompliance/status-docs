# Sistema de Gestión de Cumplimiento

Estructura jerárquica de cumplimiento con catálogos que contienen controles que generan cálculos y puntos de garantía.

## Descripción General

El Sistema de Gestión de Cumplimiento proporciona un enfoque estructurado para gestionar los requisitos de cumplimiento a través de una organización jerárquica de catálogos, controles, cálculos y puntos de garantía.

## Características Principales

- **Estructura Jerárquica**: Organizar los requisitos de cumplimiento en una jerarquía basada en catálogos
- **Gestión de Controles**: Definir y gestionar controles de cumplimiento dentro de los catálogos
- **Generación de Cálculos**: Generar automáticamente cálculos a partir de los controles definidos
- **Puntos de Garantía**: Rastrear y almacenar puntos de garantía para la verificación del cumplimiento

## Arquitectura

El sistema sigue un modelo jerárquico:
1. **Catálogos**: Contenedores de nivel superior para organizar los requisitos de cumplimiento
2. **Controles**: Reglas y requisitos de cumplimiento específicos dentro de los catálogos
3. **Cálculos**: Cálculos automatizados derivados de los controles
4. **Puntos de Garantía**: Puntos de datos que verifican el estado de cumplimiento
